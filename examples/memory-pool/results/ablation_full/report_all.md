## 实验一：页锁内存消融

消融维度：`write_memory_pool(memory_pool_id, tensor_info, *, mode=...)`

- **auto** — 根据 25 MiB 阈值自动选择 pinned / pageable（生产默认）
- **pinned** — 始终 `cudaHostRegister` + pinned DMA
- **pageable** — 跳过 `cudaHostRegister`，使用 pageable `cudaMemcpy`

*仅测试 `cpu2cuda` 场景（pinned vs pageable 在 cpu2cpu / cuda2cpu 上无 DMA 路径差异）。*
